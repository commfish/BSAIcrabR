# Add target stock to data

Add target stock based on fishery code

## Usage

``` r
add_target_stock(x, col = fishery)
```

## Arguments

- x:

  data frame

- col:

  fishery code colum, default = fishery

## Value

x with additional column 'target_stock'

## Examples

``` r
add_target_stock(x)
#> Error: object 'x' not found
```
